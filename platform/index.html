华炎魔方元数据深度解析
华炎魔方使用元数据定义对象，字段，配置，代码，逻辑和页面布局，元数据可以导入到华炎魔方中，可以在可视化界面中进行修改，也可以同步到源码进行深度开发。
  • 数据结构
    创建自定义对象，以存储贵组织独有的信息。
  • 验证规则
    验证用户在记录中输入的数据是否符合您的要求。
  • 工作流规则
    通过工作流，您可以自动化标准的内部流程，以节省整个组织的时间。
  • 自动化操作
    自动执行某些操作，例如更新字段或发送电子邮件。
  • 批准过程
    比工作流规则更进了一步，您可以指定批准记录所需的一系列步骤。
  • 统计报表
    帮助您分析和理解数据并及时采取行动。
  • 数据权限
    定义了一套企业级的权限模型，帮助用户锁定私有数据。
  • 高级业务逻辑
    使用代码实现高级业务逻辑，例如触发器、与第三方系统接口等。
零代码创建自定义应用程序
华炎魔方内置功能强大的建模引擎，可以任意配置业务字段、列表视图、页面布局，即使是不懂编程的业务人员，也能快速创建智能化、移动化的企业应用程序。
业务对象
业务对象相当于数据库中的表，用于保存业务系统中的核心数据。
列表视图
通过配置列表视图，最终用户可以快捷的切换多个查询条件。
页面布局
页面布局控制了记录的显示方式，可以按照用户权限分配不同的页面布局。
使用公式执行自动计算
华炎魔方内置了类似Excel的公式引擎，通过引入公式，可以为您的应用程序增加业务的自动化判断、计算和处理的过程。
引用相关表
可以在定义公式，引用相关表的数据。
计算与条件判断
通过计算与条件判断函数，实现相对复杂的业务逻辑。
累计汇总
自动汇总计算子表中的相关数据。
使用华炎魔方DX同步元数据
可视化配置的元数据可以和代码中的元数据双向同步，开发人员可以使用熟悉的工具编写代码，管理版本。
源码驱动开发
用你熟悉和喜爱的工具（Github、VS Code 等等）进行构建。
VS Code 插件
源码高亮显示、错误提示、代码段生成、Git版本管理
编码调试与测试
和传统企业软件开发一样，编写触发器、服务端API、表单事件等代码，满足个性化需求。
使用代码编写高级业务逻辑
华炎魔方提供了低代码开发的入口，开发人员通过编写代码，就能实现特定条件下的自动运行、截停、回滚等业务逻辑，让业务在无人值守的情况下自动运转，驱动效率提升。
触发器
当服务端执行数据增删改查时，自动触发相关代码。
操作按钮
基本的增删改按钮的基础上，为业务对象增加自定义的操作按钮。
服务端API
自定义服务端API，编写微服务，实现自动化的业务处理。
