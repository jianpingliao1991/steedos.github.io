Condition node

System can select next step automatically according to the condition in condition node.

For instance, in a ‚ÄúReimbursement‚Äù request, system can select next step according to ‚ÄúTotal Amount‚Äù. If total amount is more than 10000, requests will be submitted to general manager at first and then submitted to finance director after approval. If it is no more than 10000, requests will be directly submitted to finance director.

Conditional formulaÔºö{ Total Amount }>10000 and { Total Amount }<=10000

notice#

  • There cannot be overlapping conditions. For instance, ‚Äú{Total Amount}>=10000‚Äù and ‚Äú{Total Amount}<=10000‚Äù are wrong formulas. Because ‚Äú{Total Amount}=10000‚Äù is overlapping condition.
  • The formulas should cover all conditions. For instance, ‚Äú{Total Amount}>10000‚Äù and ‚Äú{Total Amount}<10000‚Äù are wrong formulas. Because ‚Äú{Total Amount}=10000‚Äù is omitted.

If there are wrong formulas, system cannot select the next step sometimes.

How to add a condition node?#

Please follow the next step to add a node.

  • Enter the flow design interface and click ‚ÄúDraw‚Äù button.
  • Select ‚ÄúCondition‚Äù icon and Drag it to the left side.
  • Modify step name by clicking ‚ÄúProperties‚Äù. For instance, rename it with ‚ÄúWhether Total Amount is more than 10000‚Äù.
  • Hover with your mouse over the ‚Äú?‚Äù icon and then click ‚Äú+‚Äù to add two lines and connect them with two nodes.
  • Click one line and then click on ‚Äúproperties‚Äù
  • Input the formula ‚Äú{Total Amount}>10000‚Äù in ‚Äúexecute only when‚Äù box
  • Then input the formula ‚Äú{Total Amount}<=10000‚Äù to modify properties of the other line.
