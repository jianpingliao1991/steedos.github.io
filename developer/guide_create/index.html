创建项目

#创建Steedos项目

创建Steedos项目前，请安装开发环境

创建一个名为test的空项目

npx steedos create test

#安装依赖包

cd test
yarn

#运行空项目

终端执行以下代码：

yarn start

#文件夹结构

项目创建后，您的项目文件夹内容如图如下：

其中

  • src: 项目源码
  • steedos-config.yml: 项目配置文件，可以配置数据源、插件等参数。
  • package.json: 主要用于配置本项目以来的npm包。

#注册一个新账户

使用浏览器访问地址 http://127.0.0.1:5000/ ，即可访问用户界面。 第一次使用时，数据库为空。点击“注册”，用邮箱注册一个新账户。具体操作步骤，请参照审批王-注册企业账户

#创建一个企业

首次登入系统后，输入企业名称，即可完成企业的创建。
