运行项目

#运行项目

yarn start

#使用浏览器访问

使用浏览器访问地址 http://127.0.0.1:5000/ ，即可访问用户界面。 第一次使用时，数据库为空。点击“企业注册”，可以在本地数据库中创建一家新的企业账户。

#调试项目

执行以下命令，进入调试模式。调试模式下，修改任何配置文件会自动重新启动服务，开发人员只需要刷新浏览器就能看到修改后的结果。

yarn debug
