华炎开发平台培训教程

本教程以合同管理为例，指导你如何使用华炎开发平台创建项目，配置业务对象，编程脚本，处理业务部门的各种个性化需求。

#教程3:业务对象关联

  • 创建合同object

  • 新建合同app，配置包括的业务对象
  • 重启服务运行并查看效果
