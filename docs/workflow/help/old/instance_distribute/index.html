#分发

分发，主要用于集团公司的申请单，经过一系列的审批之后，分发到下属子公司的某个流程，并自动拟写该流程的申请单。如，河北港口集团发文流程，在“拟稿人分发”步骤，选择分发到河北港口集团基层单位收文流程，分发对象选择第一分公司的文书。则在第一分公司文书的“待审核”列表里，会出现河北港口集团基层单位收文的申请单，拟写人就是第一分公司文书，申请单的正文和附件（如果配置了正文和附件）同上级部门的申请单一致。 {% video %}http://oss.steedos.com/videos/cn/文件分发.mp4{% endvideo %}

  • 申请单在哪一步可以被分发，根据实际情况由流程管理员配置。
  • 配置了分发功能的流程，在允许分发的步骤上，点击进入申请单，在申请单上方可以看到“分发”按钮。
  • 点击“分发”按钮，选择要分发的流程和分发对象。
  • 在分发对象的“待审核”里可以看到所选分发流程的申请单
  • 申请单的相应的内容会根据分发过来的申请单内容自动填写。
