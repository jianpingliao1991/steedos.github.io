数据字典

OData服务是一种自描述服务，它公开定义实体集，关系，实体类型和操作的元数据 获取对象的数据结构接口。

#接口信息

  • 请求地址：/api/v4/$metadata

  • 请求方法：GET

  • 接口说明：

    • spaceId：工作区ID
    • $metadata：固定值

  • 返回值：服务端数据结构文档

  • 示例如下： - HTTP 请求

    ```
            url 
              -X GET https://beta.steedos.com/api/odata/v4/Af8e****DqD3/$metadata
           ```
        - HTTP 响应
            ```
